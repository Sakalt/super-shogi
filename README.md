## これは何？

ブラウザで遊べる将棋です。

[このページ](https://izumo27.github.io/react-shogi/)で遊ぶことができます。

React.jsを使って実装しています。

通信機能を実装した[反転将棋](https://inversion-shogi-c121c.firebaseapp.com/)では盤面を反転することができます。

![](/public/shogi_pv.gif)

## 実装済み
- 駒の動き
- 詰み判定
- 二歩
